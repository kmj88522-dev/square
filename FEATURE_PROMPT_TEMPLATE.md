# Feature Prompt Template

Use this after Square v0.1 is working.

---

Read the project markdown files first.

Add only the following feature to Square.

Feature:

```text
DESCRIBE ONE FEATURE HERE
```

Rules:

- Keep the scope narrow.
- Do not implement unrelated roadmap features.
- Preserve existing behavior.
- Do not add heavy dependencies unless necessary.
- Provide full file contents for changed files.
- Explain how to test the new feature.

Do not implement:

- Formula, unless this feature explicitly asks for Formula.
- Graph View, unless this feature explicitly asks for Graph View.
- Calendar, unless this feature explicitly asks for Calendar.
- Document processing, unless this feature explicitly asks for Document processing.
- Google integration, unless this feature explicitly asks for Google integration.
