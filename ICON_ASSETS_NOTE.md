# Square Icon Assets Note

Square visual language:

- Minimal
- Black and white
- Square/rectangle-based
- Geometric
- Clean and app-like
- Transparent background preferred for icons

Current icon direction:

- Main app icon: two interlocking square-like frames
- Book icon: minimal book/ledger shape
- Page icon: minimal page/document outline
- SubPage icon: page with nested smaller page cue
- Block icon: simple square/cube/block unit

Recommended future handling:

- Keep generated image icons as reference.
- Later convert final icons into SVG for clean UI use.
- Use consistent stroke width and corner style.
- Prefer transparent background PNG/SVG.
- Avoid overly complex or symbolic shapes that could be misread.
